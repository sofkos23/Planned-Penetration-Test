# Mini penetrationstest – OWASP Juice Shop

## Beskrivning
Detta GitHub-repo innehåller material kopplat till ett planerat penetrationstest som genomfördes inom kursen Ethical Hacking.

Testobjektet var OWASP Juice Shop, en medvetet sårbar webbapplikation framtagen för utbildningssyfte.

## Testmiljö och omfattning
- Testet genomfördes lokalt i en isolerad miljö med hjälp av Docker  
- Endast applikationsnivån testades (Layer 7 i OSI-modellen)  
- Testningen följde OWASP:s testmetodik  

## Etiska överväganden
All testning genomfördes i en kontrollerad labbmiljö i utbildningssyfte.  
OWASP Juice Shop är avsiktligt sårbar och innehåller ingen verklig användardata.

Testningen var begränsad till definierad omfattning och inga angrepp som kunde påverka tillgänglighet eller externa system genomfördes.
